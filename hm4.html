Какие бываеют CSS селекторы

  • Универсальные селекторы
  • Селекторы по названию тега
  • Селекторы по классу
  • Селекторы по id
  • Селекторы по атрибуту
  • Селекторы потомков, или контекстные селекторы
  • Селекторы дочерние (только первые)
  • Селекторы сестринские, или соседние (расположенные на одном уровне)
  • Селекторы псевдоклассов (селекторы состояния)
  • Селекторы псевдоэлементов

* { font-size: 16px; } Будет применено для всего документа, шрифт будет 16 пикселей

h1 { color: red; } Будет применено для заголовка h1, заголовок будет красный

.cat { background-color: red; } Будет применено к элементу к которому прописан класс cat, цвет фона будет красный

#car { font-size: 35px; } Будет применено к элемену к которому прописа id car, размер шрифта будет 35 пикселей

font-family - Устанавливает семейство шрифта, которое будет использоваться для оформления текста содержимого. Список шрифтов может включать одно или несколько названий, разделённых запятой. Если в имени шрифта содержатся пробелы, например, Trebuchet MS, оно должно заключаться в одинарные или двойные кавычки.

background- Универсальное свойство background позволяет установить одновременно несколько характеристик фона. Значения могут идти в любом порядке, браузер сам определит, какое из них соответствует нужному свойству. Для подробного ознакомления смотрите информацию о каждом свойстве отдельно. Допустимо, также, указывать параметры сразу нескольких фонов, перечисляя их через запятую.

color - Определяет цвет элемента

background-image - Устанавливает фоновое изображение для элемента

margin - Устанавливает величину отступа от каждого края элемента
